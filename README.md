## CS16 Source Menu

Donnez a votre jeu Godot une interface old-school inspiree Counter-Strike, en quelques minutes.

`CS16 Source Menu` est un addon runtime independant qui apporte un menu principal complet et un systeme d'options vraiment fonctionnel, avec un rendu CS immediat.

- Theme visuel CS via `res://ressources/themes/cs16_classic.theme`
- Menu principal: `New Game`, `Options`, `Credits`, `Exit`
- Onglets options: `Controls`, `Inputs`, `Audio`, `Video`, `Game`
- Rebinding clavier + application runtime (audio, video, langue, controls)
- Sauvegarde automatique des reglages dans `user://`

Instanciez `res://addons/cs16_source_menu/scenes/source_menu_module.tscn`, connectez les signaux, et le menu est pret a l'emploi.
